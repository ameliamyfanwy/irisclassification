# IRIS CLASSIFICATION

This is a model to classify irises into one of three species based on four characteristics - sepal length, sepal width, petal length and petal width. 

![image](https://user-images.githubusercontent.com/36647468/85921061-0f8dfc00-b871-11ea-826f-67e45c5e3e3b.png)

### HOW TO RUN

You will need the following libraries:

- ```sklearn```
- ```matplotlib```
-  ```numpy```

If not installed, you can install them with this command:

```pip install scikit-learn matplotlib numpy```

### PART ONE

You can select which data to plot and produce a scatter plot

### PART TWO

You can use the model to predict the species of an iris with any input
